About CryptoTicker

CryptoTicker is a simple and responsive web application that allows users to monitor live cryptocurrency prices using real-time market data.

Built with HTML, CSS and Vanilla JavaScript, this project demonstrates API fetching, asynchronous programming, responsive UI, accessibility, and modern web development practices.
